# Shalla Wellness Center Website

This is a modern, responsive website built for Shalla Wellness Center.

## Tech Stack
- Next.js
- React
- Tailwind CSS
- Framer Motion

## Features
- Multi-page layout (Home, About, Services, Contact)
- Smooth animations and transitions
- Responsive design
- Clean and scalable structure
